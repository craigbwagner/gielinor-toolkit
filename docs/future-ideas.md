# Future Ideas

Features and integrations that don't have a milestone yet. These may be added to the roadmap as the project matures.

## Ely.gg Integration
- Street prices for rare items (partyhats, dyes, hero items) that don't trade reliably on the GE
- No public API currently — would need scraping or reaching out for API access
- Would add a third price source alongside Weird Gloop and GE guide prices

## PvME Boss Revenue Data
- PvME (pvme.io) maintains calculated GP/kill values for commonly farmed bosses
- Data is sourced from the RS Wiki and available on their GitHub
- Could use this to show expected GP/hr or compare personal loot to average

## Session Tracking
- Start/stop a PvM session tied to a specific boss
- Track kills per session, wealth per session, kills/hr
- Session history log

## Collection Log Tracker
- Dedicated view for tracking collection log completion across all bosses
- Pre-populated checklists based on RS Wiki data
- Completion percentage per boss and overall

## RS Wiki Drop Table Integration
- Pull structured drop tables from the RS Wiki for each boss
- Display expected drop rates alongside actual drops
- Auto-populate the drop logger's item list from the boss's known drop table

## Desktop Notifications
- Alert when a goal is completed
- Price alerts for watched items (e.g. "Leng artefact dropped below 1B")
- Tauri supports native desktop notifications

## Alt1 Toolkit Integration
- Alt1 is a popular RS3 overlay toolkit
- Could potentially read screen data for auto-detection of drops or boss kills
- Would eliminate manual logging for some scenarios

## Import/Export
- Export all data as JSON for backup
- Import data from other tracking tools (RuneApps drop logger, etc.)

## Multiple Character Support
- Track boss data and goals for multiple RS3 accounts
- Switch between characters in the UI
